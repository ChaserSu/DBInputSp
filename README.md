# DBInputSp
一个双拼正反查工具，协助输入法的开发和日常使用
# 主要有三个作用
- 1、正查：把文字转化为双拼编码
- 2、反查：把双拼编码转换成拼音
- 3、查表：输出完整双拼对应键位表
